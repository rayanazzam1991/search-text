# Search Text

## Clone

Clone from public repository

```bash
# git
git clone https://github.com/rayanazzam1991/search-text.git
cd search-text

```


## Setup

Make sure to install the dependencies:

```bash
# npm
npm install

# pnpm
pnpm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev

# bun
bun run dev
```
